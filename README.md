# DeepLabCut_pose_estimation

A DeepLabCut-based computer vision pipeline for behavioral video analysis, markerless pose estimation, movement tracking, and quantitative feature extraction in fruit fly locomotion studies.

# Overview

This repository contains workflows for automated behavioral imaging analysis using DeepLabCut and Python-based computer vision pipelines. The project focuses on extracting quantitative movement and pose-related features from high-frame-rate fruit fly behavioral videos.

The workflow supports markerless tracking, locomotion analysis, behavioral quantification, gait analysis, and downstream statistical and machine learning analysis.

## Objectives

* Markerless pose estimation
* Fruit fly behavioral tracking
* Movement quantification
* Automated feature extraction
* Gait and locomotion analysis
* Behavioral classification
* Computer vision-based behavioral analysis

## Workflow

1. Video acquisition and preprocessing
2. ROI preparation and frame extraction
3. DeepLabCut pose-estimation pipeline
4. Tracking quality control
5. Coordinate extraction and filtering
6. Feature extraction and behavioral quantification
7. Statistical and machine learning analysis
8. Visualization and interpretation
